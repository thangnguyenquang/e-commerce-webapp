# e-commerce-webapp

*Dự án vẫn đang trong quá trình hoàn thành

Tools and technologies used:
- Oauth2
- Spring Security
- Spring Boot
- Spring MVC
- Spring Data JPA ( Hibernate)

- Thymeleaf
- SpringToolSuite4
